# Depths-Of-Endor-save-editor

valid for version 5.7.0

------------
Requirements
------------

1. python 3.8+

2. pycryptodome



----------
How to Use
----------

1. Start decryption in terminal:

   python decode.py "encryptedFile.dat" "decryptedFile.json"

2. Open the resulting file and change the necessary parameters

3. Encryption of edited storage:

   python encode.py "decryptedFile.json" "encryptedFile.dat"
